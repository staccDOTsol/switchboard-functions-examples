# Sbv2-Scripts

scripts used within the repo

## Setup

```bash
pnpm install
```

## Commands

### `simulate`

<pre>
<b>DESCRIPTION:</b>
Simulate an OracleJob or directory against the task runner

<b>USAGE:</b>
sb-tools simulate [JSON_FILE|YAML_FILE|FILE_DIRECTORY]

<b>EXAMPLE:</b>
sb-tools simulate my_feed.json
sb-tools simulate my_feed.yaml
</pre>
